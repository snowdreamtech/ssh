# ssh

[![ssh](http://dockeri.co/image/snowdreamtech/ssh)](https://hub.docker.com/r/snowdreamtech/ssh)

Docker Image packaging for ssh.

(amd64, arm32v6, arm32v7, arm64v8, i386)

## Usage

```bash
docker run --name ssh -ti --rm snowdreamtech/ssh /bin/sh
```
